# VectorIMG
A high-performance Python engine designed for efficient image manipulation, format transformation, and color space optimization (pixelation, blurring, and color inversion). By leveraging NumPy’s compiled C-backend, the program executes complex data operations using zero-loop vectorization, ensuring high-speed processing of large-scale image data.
